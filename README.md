# Hi there, I'm Nguyen Anh Kiet 👋

### 🧑‍💻 Who are you?
I am a final-year Computer Engineering Technology student at the Industrial University of Ho Chi Minh City (IUH). I have a strong passion for hardware architecture, microchips, and embedded systems. I enjoy bridging the gap between hardware and software, turning logic designs and C code into real-world applications.

### 🚀 What are you currently working on?
I am currently dedicating my time to building large-scale VLSI, FPGA, and Embedded C projects to gain hands-on experience for my upcoming career in the tech industry. 

Some of my key projects include:
* **Synchronized Traffic Light Control System:** Developed an embedded system using **PIC18F4550**, integrated with a custom **C# Windows Forms GUI**. The system features dual-control modes (physical buttons and software interface) with precise initialization and reset states during mode transitions.
* **Smart Automated Drying Rack:** Designed a logic-based hardware circuit using ICs, LDRs, and rain sensors to automate a motorized rack, implementing hardware limit switches to ensure strict motor safety overrides.
* **Digital Signal Processing (DSP) Implementations:** Designed and simulated FIR/IIR filters, performed spectral analysis, and worked with audio signal processing using **MATLAB** and the **TMS320C6416 (DSK6416)** hardware kit.
* **FPGA & VLSI Exploration:** Working with **Verilog**, hardware-software co-design using **Nios II** processors, and Qsys address mapping.
